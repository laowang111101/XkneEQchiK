# Vue3+Element Plus 商城后台管理系统 java278

## 项目概述

本项目是基于 Vue3 和 Element Plus 的商城后台管理系统，采用前后端分离的设计模式。系统具备商品管理、商品分类、个人中心等基础功能模块，界面美观，操作简便，适合初学者学习和使用。

## 技术栈

- **前端框架：** Vue3
- **UI 库：** Element Plus
- **构建工具：** Vite
- **状态管理：** Pinia
- **数据模拟：** Mock.js 和 Axios

## 功能模块

- **首页：** 显示数据统计
- **商品管理页：** 包含商品的添删改功能
- **商品分类页：** 管理商品分类信息
- **个人中心：** 用于查看和修改个人信息
- **登录页面：** 系统登录入口

## 系统角色

- 管理员：负责后台系统的操作与管理

## 运行环境

## 部署步骤

1. 确保本地已安装 Node.js 和 npm
2. 使用 VScode 打开项目文件夹
3. 在项目根目录执行 `npm install` 安装依赖
4. 运行 `npm run dev` 启动开发服务器
